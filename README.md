## Track Expenses

### Data Model

**Accounts** - where our money is allocated

* Name
* Description
* Initial Value

**Transactions**

* Name
* Description
* Amount
* Source (null if income/interest)
* Destination (null if expense/fees)
* Executed At
* Category Id

**Categories**

* Name
* Description
